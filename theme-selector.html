How to Apply a Theme

  1. Preview themes: Click on any theme card above to open it in a new browser tab and see how it looks.
  2. Choose your favorite: Browse through all the options and decide which style best represents you.
  3. Apply the theme: To permanently apply a theme, edit your index.html file and change line 20:
    • Original: <link rel="stylesheet" href="styles/main.css">
    • Gradient Modern: <link rel="stylesheet" href="styles/theme-gradient.css">
    • Minimalist Monochrome: <link rel="stylesheet" href="styles/theme-minimal.css">
    • Warm Academic: <link rel="stylesheet" href="styles/theme-academic.css">
    • Tech/Futuristic: <link rel="stylesheet" href="styles/theme-tech.css">
  4. Switch anytime: You can easily switch between themes by changing that one line in your HTML file.
